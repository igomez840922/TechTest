
Crear un sistema de Registro de Productos:
Los mismos deben incluir los siguientes campos

- Modelo (obligatorio)
- Nombre (obligatorio)
- Descripcion (opcional)
- Precio (obligatorio)
- Una Foto (obligatorio)

- En una vista principal donde muestre todos los productos del sistema, y posibilidad de Agregar Nuevo, Editar, Eliminar (CRUD completo)
- Posibilidad de Exportar via Pdf o Excel

Requerimientos:
1- Utilizar Entity Framework Core para manejo de BD(SQL o SQL Express) (obligatorio)
2- Autenticacion y Autorization (Login y Password) para entrar a la app.
Utilizar Identity. (obligatorio)
3- Manejo de Seguridad de APIs, preferiblemente JWT (obligatorio)
4 - Manejo de excepciones generales (obligatorio)
5- Usar en la UI MudBlazor si lo conoce. (Opcional)
